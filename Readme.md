# hedgehog test

Very simple example on how to run `qemu in hedgehog <https://github.com/EljakimHerrewijnen/qemu>`_ mode, which is a python wrapper around qemu.

The idea is to instrument qemu as you would use unicorn but now synced with the main qemu branch instead of an older fork.
It should also better support exception levels and such.

## Run
After install hedgehog, do:

```
python3 main.py
```
